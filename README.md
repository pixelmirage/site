# mertkagancetin.com

İzmir kira avukatı Av. Mert Kağan Çetin'in resmi web sitesi.

**[mertkagancetin.com](https://mertkagancetin.com)**

## Hakkında

[Mert Kağan Çetin Hukuk Bürosu](https://mertkagancetin.com), İzmir'de kira hukuku, tahliye davaları, kira uyuşmazlıkları ve gayrimenkul hukuku alanlarında hukuki danışmanlık ve dava takip hizmetleri sunmaktadır.

### Hizmetler

- Kira sözleşmesi hazırlama ve inceleme
- Tahliye davaları (iki haklı ihtar, ihtiyaç nedeniyle tahliye)
- Kira tespit ve uyarlama davaları
- Zorunlu arabuluculuk süreç yönetimi
- Gayrimenkul hukuku danışmanlığı

## Teknolojiler

- [Next.js](https://nextjs.org/) - React framework
- TypeScript
- MDX - Blog içerik yönetimi
- Tailwind CSS - Stil
